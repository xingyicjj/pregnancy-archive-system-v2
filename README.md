# 孕期档案管理系统

一个基于 React + TypeScript + Tailwind CSS 的智能孕期健康档案管理系统，支持医疗报告上传、AI 分析和健康建议。

## ✨ 功能特点

- 📱 **响应式设计** - 完美适配手机和桌面端
- 🤖 **AI 智能分析** - 自动识别和分析医疗报告
- 📊 **健康档案** - 完整的孕期健康记录管理
- 🔍 **报告搜索** - 快速查找历史报告
- 💡 **个性化建议** - 基于用户档案的健康建议

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 部署到 Vercel

1. 将项目推送到 GitHub
2. 访问 [Vercel](https://vercel.com)
3. 导入 GitHub 仓库
4. 自动部署完成

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **图标库**: Lucide React
- **图表库**: Recharts
